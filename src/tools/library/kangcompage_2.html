도서특징
C++ 프로그래밍의 모든 것을 담은 바이블

살아 숨쉬는 디텔 라이브 코드(LIVE-CODETM) 실무 예제로 풀어낸 C++ 프로그래밍

세계에서 가장 널리 쓰이는 C++ 교재로 인정받아온 『C++ HOW TO PROGRAM』의 4판인 이 책은 C++가 지닌 비상한 기능을 설명하고, OMG(Object Management Group)의 UML(Unified Modeling Language)을 이용한 객체 지향 설계(OOD)와 구체적인 구현 사례를 제시하며, CGI를 이용한 멀티티어(n-tier) 웹 애플리케이션 개발 등, C++를 이용한 객체 지향 프로그래밍의 기초에서 일반 프로그래밍의 완성까지를 한 권에 담아내고 있다.


이 책의 특징

  • LIVE-CODE 예제 수백 개를 정확한 결과를 보여주는 캡처 화면과 함께 제공
  • 추가 학습 의욕을 돋궈주는 인터넷 및 웹 리소스를 일목요연하게 정리
  • 수백 개에 이르는 팁, 조언, 습관, 주의 등을 한 눈에 띄는 아이콘으로 정리
